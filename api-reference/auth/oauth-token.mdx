description: "OAuth2 client-credentials flow for legacy integrations."
openapi: "POST /oauth/token"
mode: "wide"
---

Most integrations should use the standard token endpoint: [Get Authentication Token](/api-reference/auth/auth-token).

If your account is configured for OAuth2, use the **client_credentials** flow on this endpoint. Contact Waftpay support if you are unsure which token flow applies to your account.
