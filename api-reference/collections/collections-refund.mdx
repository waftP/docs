---
title: "Initiate Refund Reversal (Query)"
description: "Query the status of a refund reversal by request UUID."
openapi: "POST /refund-reversal"
---

<Note>
    This endpoint **does not create** a refund; it **queries the status** of an already-initiated refund reversal using a <code>request_uuid</code>.
</Note>

## Prerequisites

- **Bearer token** from `POST /authentication-service/v1/api/generate-token` (Refunds/Reversals product scope)
- **Content-Type**: `application/json`

---

## Required headers

```http
Authorization: Bearer <token>
Content-Type: application/json
