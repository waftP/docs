---
title: "Check Refund/Reversal Status"
description: "Query refund or reversal status using a request UUID."
openapi: "POST /refund-reversal"
---

Use the refund/reversal status endpoint and provide the `request_uuid` you received when the refund/reversal was created.

```json
{
  "request_uuid": "A333RTYUT"
}
```

See [Refund/Reversal Status](/api-reference/collections/collections-refund) for full details and response examples.
